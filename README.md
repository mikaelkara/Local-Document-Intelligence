# Local-Document-Intelligence-
Bu proje, bilgisayarda bulunan belgeleri otomatik olarak analiz eden, sınıflandıran, yeniden adlandıran ve düzenleyen akıllı bir belge yönetim sistemidir. Sistem tamamen offline çalışır; tüm işlemler kullanıcı cihazında gerçekleştirilir ve hiçbir veri dışarıya gönderilmez.
